# Testes E2E com Playwright no Conduit

Este repositório utiliza o projeto **Conduit RealWorld Example App** como aplicação base open source.

Aplicação original:  
https://github.com/TonyMckes/conduit-realworld-example-app

A autoria da aplicação base pertence aos mantenedores do projeto original.  

O objetivo deste repositório é demonstrar minha implementação de testes end-to-end com **Playwright**.
